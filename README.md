## Como conectar com o banco de dados
Após rodar a aplicação, acesse a página: http://localhost:8080/h2-console

- URL: `jdbc:h2:mem:deliveries-db`
- Usuário: `user`
- Senha: `password`

Depois de preencher os campos, clique em `Connect`
